Homework: Final Project!

The requirements:

  • You must use ES6 and transpile your code to ES5 using Babel.
  • Use a CSS pre-processor (like Sass or Less) OR inline styles in React OR CSS Modules.

Incorporate at least 3 of the following:

  • Load content from an API (either a 3rd party API or one that you build)
  • SVG
  • React
  • CSS Grid
  • Flexbox
  • Websockets
  • CSS Animations
  • Update the DOM based on user input
  • Local Storage
  • Express
  • Webpack

What is NOT allowed:

  • No 3rd party styling frameworks like Bootstrap, or Foundation.

Submission & Presentations:

You must submit your project as a github repo before the beginning of next class.

Prepare a 5-8 minute presentation of your final project to present in the last class (7/11), including a walkthrough of how the code works.

Everyone who completes and presents a final project gets to attend the class lunch.
