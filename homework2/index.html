Instructions

Build a form that will do the following when submitted:

  • If valid, log 'VALID' to the console.
  • If invalid, show an alert box to the user with an error message. You can either show a generic error message, or for extra credit, show an error message with specific details about what went wrong.

This form should have at least:

  • One text field
  • One select field
  • Two other input fields (any type you want)

Validation rules:

  • All 4 fields are required.
  • For your text field, nothing else except a value of 5 will be accepted.
  • For your select field, nothing else except a value of "frogs" will be accepted.

Your page should:

  • use CSS to style the contents
  • use a grid system (feel free to use Bootstrap, or Foundation if you want)
  • use at least one media query (that you write yourself)
