Homework

Using what you now know about JavaScript, HTML, and CSS, build a clock. The clock can have a hardcoded time, or show a time that changes, it's completely up to you. However:

  • Include a button that, when clicked will change the time on the clock.
  • Use a CSS pre-processor. (Either Sass, Less, or something else of your choice).
  • Use Browserify to combine at least 2 javascript files into a single one that is loaded on the page.
  • Do not use an image of a clock:). Build your clock using HTML, CSS, and JavaScript.
