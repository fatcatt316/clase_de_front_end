JokeDatabase.gov

Official Joke Database of the U.S.A.

Build an app that does the following:

  • Has a list of quotes/sentences/jokes/whatever you want.
  • Has a button to add user input as a new item to the list.
  • Has a button to clear the list.
  • The list is saved to local storage so existing values will load when you refresh the page (in Chrome) without clearing your cache.
  • Uses webpack to bundle your JavaScript.
